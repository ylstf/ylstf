# 识字闯关星球 V3

V2 增强版纯网页应用。保留了 V2 的温暖星球视觉风格，新增多项功能。

## 相比 V2 的新功能

- **200 题快速估算**（原 60 题）— 更稳定的识字量评估
- **撤回** — 快速估算和分组闯关都支持撤回最近 2 步
- **多孩子管理** — 起昵称切换，每个孩子独立存储
- **错字本** — 查看不认识的字，可逐个或全部重测
- **分享报告** — Canvas 绘制报告图，一键保存到本地
- **分组列表** — 独立页面浏览所有 25 组进度

## 使用

直接打开 index.html 即可使用，无需服务器。

推荐部署到 GitHub Pages / Gitee Pages / CODING Pages 后通过链接或二维码分享给家长。

## 字库

2500 常用字，分 10 个难度等级，来源：《现代汉语常用字表》。

## 技术

纯 HTML + CSS + Vanilla JS，无框架无构建，localStorage 存储。
